# STAGE ONE — деплой за 3 шага

Прототип: витрина 15 айдолов → выбор стиля песни/языка/локации/танца → генерация клипа с вокалом через **Seedance 2.0 (fal.ai)**.

## Что нужно
- Аккаунт на [vercel.com](https://vercel.com) (бесплатный)
- Новый API-ключ **fal.ai** (старый, что был в чате, — перевыпусти!)

## Шаг 1. Залей проект
Вариант проще — через сайт Vercel:
1. Запакуй эту папку в zip (кроме `node_modules`).
2. На vercel.com → **Add New → Project → Deploy** — перетащи папку/zip.
   Либо через терминал: `npm i -g vercel && vercel` в этой папке.

## Шаг 2. Вставь ключ (ГЛАВНОЕ)
В Vercel: **Project → Settings → Environment Variables**
- Name: `FAL_KEY`
- Value: *твой новый ключ fal.ai*
- Сохрани → **Redeploy** (Deployments → ⋯ → Redeploy).

Ключ живёт только здесь. В коде его нет.

## Шаг 3. Открой и проверь
Открой выданный Vercel адрес (`*.vercel.app`):
1. Выбери айдола.
2. Задай стиль песни, язык, локацию клипа, стиль танца.
3. **Создать клип** → через 30–90 сек появится видео с вокалом.

---

## Важно
- Каждая генерация тратит баланс fal.ai (~$0.37 за клип 5 сек 720p). Пополни баланс в кабинете fal.ai.
- Один клип = один айдол (image-to-video оживляет один портрет). Группа из нескольких — отдельный этап (reference-to-video).
- Видео несёт невидимый C2PA-водяной знак «AI-generated» в метаданных.
- Если генерация падает: проверь, что `FAL_KEY` задан и на балансе есть деньги.

## Структура
```
api/generate.js   — serverless: строит промпт, зовёт Seedance 2.0
public/index.html — витрина + персонализация + плеер
public/idols.json — данные 15 айдолов
public/idols/     — 15 портретов
vercel.json       — таймаут функции 300с
```
